Student Management System

A simple **student** management application built using Python (Tkinter).  
It helps manage student records like name, roll number, course, and marks.
## Pre requisite 
- install XAMPP server and activate phpmyadmin for database connection.
- Keep the table enteries name same in databse as written in query .
## Features
- Add, update, delete student records(CRUD)
- Search/filter students
- Store data in <file / SQLite / MySQL> database
- Simple graphical user interface
## Tech Stack
- Language: Python 3.x
- GUI: Tkinter
- Database: <SQLite / MySQL>
## Future Improvements
- User authentication (admin login)
- Export reports (PDF/Excel)
- Better UI design and validation
